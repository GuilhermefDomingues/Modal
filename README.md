# CSS Modal

Projeto desenvolvido com HTML e CSS que implementa uma janela modal utilizando apenas CSS. A abertura e fechamento do modal são controlados através da pseudo-classe `:target`, eliminando a necessidade de JavaScript.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Modal sem JavaScript
- Abertura através de link
- Fechamento por botão
- Exibição de imagem
- Exibição de informações do usuário
- Layout centralizado
- Design moderno

Praticar a criação de modais utilizando apenas HTML e CSS, explorando técnicas modernas para exibição de conteúdo sem dependência de JavaScript.
